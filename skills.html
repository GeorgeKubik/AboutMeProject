JavaScript
Я знаком с основными концепциями и структурами языка, включая переменные, типы данных, функции, условные операторы, циклы и обработку ошибок. Я умею использовать JavaScript для взаимодействия с документом, изменения структуры, стиля и содержания веб-страницы в реальном времени. Я знаю, как работать с событиями в JavaScript для создания интерактивных веб-приложений, включая обработку пользовательских действий, таких как клики, наведение и ввод данных. Я имею представление о том, как отправлять асинхронные запросы к серверу и работать с API для получения и отправки данных. Мои текущие знания в JavaScript являются фундаментом. Я осознаю, что передо мной еще много всего, что нужно изучить и освоить, включая более глубокое понимание JavaScript.

React
Я выбрал библиотеку React после сравнения её с Vue и Angular, исходя из нескольких важных выводов. Во-первых, React оказался более востребованным на рынке по сравнению с другими фреймворками, что обусловлено его гибкостью и динамичностью в разработке. Работа с React предоставляет уникальную возможность не только создавать высокоадаптивные пользовательские интерфейсы, но и разрабатывать или совершенствовать собственные стратегии разработки. Это открывает простор для инноваций и позволяет адаптировать процесс под конкретные задачи и цели проекта, делая каждое приложение уникальным и эффективно решающим поставленные задачи.
